# Development Setup


## Clone the repository
```shell
git clone https://github.com/0xdevsachin/react-native-movie-app.git

```
## Add Api in `requests.js` file
```shell
const API_KEY = 'Your_Api_key'
```

## Install Dependencies

```shell
npm install
```

## Start Project
```shell
npm start
npm run android (for android)
npm run ios (for ios)
```
